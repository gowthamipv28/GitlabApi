####Create a test automation framework for covering the following product requirements :

1. An existing gitlab user is able to sign in to gitlab.com
2. An existing user is able to create a new private project on gitlab
3. An existing user is able to add at-least 1 member to the project as a developer
4. An existing user is able to add a new variable to the project and delete it.
5. An existing user is able to delete the project

###Instruction to run the code

####`./gradlew clean build`
